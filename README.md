# SpiderLan SAT Ops

Starlink-aware NTN operations platform for RF analysis, trust scoring, handover intelligence, incident logging, and integrated threat visualization.

## Features
- Starlink satellite visibility and ranking
- RF site suitability scoring
- Security and trust scoring
- Handover prediction
- Integrated RF + threat map
- Incident logging
- CloudRF integration

  
## Teh Stack
- Python
- Skyshield
- RF Analysis
- Streamlit

  
## Run
```bash
pip install -r requirements.txt
streamlit run spiderlan_sat_ops.py
